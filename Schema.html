Class: Schema

Schema

Schema's responsibility is tracking the set of Model classes used in the database. To include your model in that set, Schema offers Schema#register and a shortcut Schema#define methods.

Schema also handles starting a Session with Schema#from.

reducer() → {function}

Returns a reducer function you can plug into your own reducer. One way to do that is to declare your root reducer:

function rootReducer(state, action) {
    return {
        entities: schema.reducer(),
        // Any other reducers you use.
    }
}
