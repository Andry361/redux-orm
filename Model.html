Class: Model

Model

The heart of an ORM, the data model. The static class methods manages the updates passed to this. The class itself is connected to a session, and because of this you can only have a single session at a time for a Model class.

An instance of Model represents an object in the database.

To create data models in your schema, subclass Model. To define information about the data model, override static class methods. Define instance logic by defining prototype methods (without static keyword).
