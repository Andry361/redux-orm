Class: EntityManager

EntityManager

A class that manages an entity tree branch.

The manager shares the following methods with QuerySet:

  • getPlainEntities
  • all
  • filter
  • exclude
  • orderBy
  • exists
  • first
  • last
  • at
  • delete
  • update
